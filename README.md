# Mixtape Haven

Your private, ad-free music sanctuary. Rediscover the art of the mixtape.

## Development

This site is built with Jekyll and Tailwind CSS.

### Prerequisites

- Ruby 3.1+
- Node.js 20+
- Bundler

### Local Development

1. Install dependencies:
```bash
bundle install
npm install
```

2. Build CSS and start Jekyll server:
```bash
npm run serve
```

3. Visit `http://localhost:4000`

### Build for Production

```bash
npm run build
```

## Deployment

This site automatically deploys to GitHub Pages via GitHub Actions when you push to the main branch.

## Tech Stack

- **Jekyll** - Static site generator
- **Tailwind CSS** - Utility-first CSS framework
- **GitHub Pages** - Hosting
- **GitHub Actions** - CI/CD
