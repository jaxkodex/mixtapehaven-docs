# Image Assets

This directory should contain the following image assets for optimal SEO and branding:

## Required Favicon Images

- **favicon-16x16.png** - 16x16px favicon for browsers
- **favicon-32x32.png** - 32x32px favicon for browsers
- **apple-touch-icon.png** - 180x180px icon for iOS devices

## PWA Icons

- **android-chrome-192x192.png** - 192x192px icon for Android
- **android-chrome-512x512.png** - 512x512px icon for Android

## Social Media Images

- **og-image.png** - 1200x630px Open Graph image for social media sharing (Facebook, LinkedIn, Twitter)
- **logo.png** - High-resolution logo for Schema.org structured data

## Image Guidelines

### Favicon & Touch Icons
- Use the Mixtape Haven logo/brand colors
- Ensure the icon is recognizable at small sizes
- Use transparent background for PNG files

### Open Graph Image (og-image.png)
- Dimensions: 1200x630px (recommended by Facebook/Twitter)
- Include branding elements
- Add tagline: "Your private, ad-free music sanctuary"
- Use brand colors (black background with magenta/purple accents)
- Ensure text is readable when scaled down

### Creating the Images

You can use design tools like:
- Figma
- Adobe Photoshop/Illustrator
- Canva
- Favicon generators (https://realfavicongenerator.net/)

## Note

Until these images are created, the site will still function but may show broken image warnings in the browser console. Social media previews will fall back to defaults.
