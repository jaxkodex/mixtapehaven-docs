## Tips for Best Results

### When using these prompts:

1. **For DALL-E / ChatGPT:**
   - Copy the prompts as-is
   - Request "high quality" or "professional" output
   - Ask for variations if needed

2. **For Midjourney:**
   - Add parameters: `--ar 1:1` for square icons
   - Add: `--ar 40:21` for og-image (1200x630)
   - Add: `--style raw` for more literal interpretation
   - Add: `--v 6` for latest version
   - Example: `[prompt above] --ar 1:1 --style raw --v 6`

3. **For Stable Diffusion:**
   - Add negative prompt: "blurry, low quality, distorted, amateur, cluttered"
   - Recommended settings: Steps 50+, CFG 7-9
   - Use LoRA models for "neon" or "logo design" if available

4. **Post-Processing:**
   - Resize to exact dimensions needed (16x16, 32x32, 180x180, 192x192, 512x512, 1200x630)
   - Optimize PNGs for web (use tools like TinyPNG or ImageOptim)
   - Ensure transparent backgrounds for favicon/icon files
   - Test favicon visibility on both light and dark browser themes
